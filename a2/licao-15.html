Warm-up: Asking for Help

Friend A:

"I'm so stressed and tired lately. What should I do?"

Friend B:

"You should get some rest. And you shouldn't work on weekends."

Friend A:

"Should I see a doctor?"

Friend B:

"Yes, you should. It's important to take care of yourself."

Grammar: Should & Shouldn't

Should (Conselho +)

Use para dizer que algo é uma boa ideia:

  • "You should drink more water."
  • "He should study for the test."

Shouldn't (Conselho -)

Use para dizer que algo NÃO é uma boa ideia:

  • "You shouldn't eat so much junk food."
  • "They shouldn't go out in the rain."
