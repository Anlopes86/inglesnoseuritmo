Grammar focus

Understand the pattern

Exemplos com tradução para revisar sozinho.

Simple rule

  • Play / go / do: Observe a ordem da frase em ingles: sujeito + verbo + complemento. Depois troque uma palavra para criar sua propria frase.
  • Can for sports ability: Observe a ordem da frase em ingles: sujeito + verbo + complemento. Depois troque uma palavra para criar sua propria frase.
  • Present simple with routines: Use o present simple para rotinas e fatos; com he/she/it, normalmente acrescente -s ao verbo.
