Grammar focus

Understand the pattern

Exemplos com tradução para revisar sozinho.

Simple rule

Home, there is/are e preposicoes de lugar

Use there is para uma coisa e there are para duas ou mais. Use preposicoes para dizer onde os objetos ficam.

UseStructureExamplePortuguês
SingularThere is + a/an + singular nounThere is a sofa in the living room.Ha um sofa na sala.
PluralThere are + number/plural nounThere are two chairs.Ha duas cadeiras.
Placenoun + be + preposition + nounThe lamp is next to the bed.A luminaria fica ao lado da cama.
  • Perguntas: Is there a sofa? Are there two chairs?
  • Preposicoes uteis: in, on, under, next to, in front of, behind.
  • Use this is para apresentar um comodo: This is my bedroom.

There is / there are

Use this pattern to say what exists in a place. Check if the noun is singular or plural before choosing is or are.

This is my apartment.

Tradução: Este é meu apartamento.

There is a sofa in the living room.

Tradução: Há um sofá na sala de estar.

There are two chairs.

Tradução: Há duas cadeiras.

Prepositions of place

Use these words to show location or movement. Say the object first, then the place phrase.

The lamp is next to the bed.

Tradução: A luminária fica ao lado da cama.

The kitchen is small.

Tradução: A cozinha é pequena.

Where is the bathroom?

Tradução: Onde fica o banheiro?

Usage situations

Teacher guidance

  • Model natural intonation first; the learner repeats only one line at a time.
  • Assign roles A and B, then switch roles after the first reading.
  • Ask quick comprehension questions: "Este é meu apartamento." and "Há um sofá na sala de estar."
  • Feedback focus for this lesson: Home, there is/are e preposicoes de lugar. Correct one priority, then let the learner personalize a line for showing an apartment.
