Grammar focus

Understand the pattern

Exemplos com tradução para revisar sozinho.

Simple rule

  • Comparative + than: Comparativos comparam duas coisas: bigger, smaller, better, worse ou more expensive.
  • Superlative with the: Observe a ordem da frase em ingles: sujeito + verbo + complemento. Depois troque uma palavra para criar sua propria frase.
  • More + long adjectives: Observe a ordem da frase em ingles: sujeito + verbo + complemento. Depois troque uma palavra para criar sua propria frase.

Comparative + than

Observe o padrão em frases curtas. Repita em voz alta e depois troque uma palavra para criar uma frase nova.

São Paulo is bigger than my city.

Tradução: São Paulo é maior do que a minha cidade.

This restaurant is cheaper.

Tradução: Este restaurante é mais barato.

English is easier now.

Tradução: O inglês está mais fácil agora.

Superlative with the

Observe o padrão em frases curtas. Repita em voz alta e depois troque uma palavra para criar uma frase nova.

My phone is better.

Tradução: Meu celular é melhor.

This is the best option.

Tradução: Esta é a melhor opção.

Which city is larger?

Tradução: Qual cidade é maior?

More + long adjectives

Observe o padrão em frases curtas. Repita em voz alta e depois troque uma palavra para criar uma frase nova.

The train is faster than the bus.

Tradução: O trem é mais rápido do que o ônibus.

This bag is more expensive.

Tradução: Esta bolsa é mais cara.

Today is colder than yesterday.

Tradução: Hoje está mais frio do que ontem.
