Grammar focus

Understand the pattern

Exemplos com tradução para revisar sozinho.

Simple rule

Clima, it is e there is/are

Use it is para falar do clima. Use there is/are para dizer que existe algo em um lugar ou momento.

UseStructureExamplePortuguês
WeatherIt is + adjectiveIt is sunny.Esta ensolarado.
Seasonin + season/monthIt is cold in winter.Faz frio no inverno.
ExistenceThere is/are + nounThere are clouds.Ha nuvens.
  • Em ingles, usamos it para clima: It is rainy.
  • Use in para estacoes e meses.
  • Pergunta comum: What is the weather like?

Usage situations

Teacher guidance

  • Model natural intonation first; the learner repeats only one line at a time.
  • Assign roles A and B, then switch roles after the first reading.
  • Ask quick comprehension questions: "Como está o tempo?" and "Está ensolarado hoje."
  • Feedback focus for this lesson: Clima, it is e there is/are. Correct one priority, then let the learner personalize a line for talking about the weather.
