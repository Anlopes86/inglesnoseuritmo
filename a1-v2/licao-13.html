Grammar focus

Understand the pattern

Exemplos com tradução para revisar sozinho.

Simple rule

  • It as weather subject: Observe a ordem da frase em ingles: sujeito + verbo + complemento. Depois troque uma palavra para criar sua propria frase.
  • There is / are for weather: Use there is para uma coisa e there are para duas ou mais coisas.
  • Comparatives: hotter / colder: Comparativos comparam duas coisas: bigger, smaller, better, worse ou more expensive.
